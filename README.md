# Instal·lació d'Odoo al Servidor (On-Premise)

Aquest repositori conté el script d'automatització per a la instal·lació i configuració d'Odoo en un entorn de servidor. Aquest treball forma part de la pràctica **E1.2** del mòdul.

## 🚀 Contingut
* **script_instalacio.sh**: Script en Bash que realitza la instal·lació completa de les dependències, PostgreSQL i el propi servei d'Odoo.

## 🛠️ Com executar el script
Per utilitzar el script correctament, segueix aquests passos des de la terminal del servidor:

1. **Donar permisos d'execució:**
   \`\`\`bash
   chmod +x script_instalacio.sh
   \`\`\`

2. **Executar el script com a superusuari:**
   \`\`\`bash
   sudo ./script_instalacio.sh
   \`\`\`
